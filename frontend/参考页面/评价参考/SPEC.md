### 1.1 顶部导航栏
- 高度: 56px
- 背景: linear-gradient(90deg, #0a1628 0%, #1a3a5c 50%, #0a1628 100%)
- 左侧: 角色切换按钮组（教师/学生），选中状态高亮
- 中间: 系统标题"低空应急智慧评价系统" + 副标题"应急物资低空智慧运输课程"
- 右侧: 实时时钟（每秒更新）

### 1.2 左侧边栏
- 宽度: 260px
- 背景: #0d1f35
- 教师视角: 6个小组折叠面板，每组显示5名学生，点击选中
- 学生视角: 只显示当前登录学生
- 学生项: 头像(首字母)+姓名+角色标签+总评分数徽标

### 1.3 主区域Tab
- Tab1 "评价操作": 表单区域，包含所有可编辑的评价项
- Tab2 "数据大屏": 全屏数据可视化面板
- Tab3 "个人画像": 学生个人详细画像

### 1.4 底部状态栏
- 高度: 32px
- 显示: 最后更新时间、AI服务在线状态指示器

## 3. 计算逻辑规格

### 3.1 过程性评价计算（满分100）
```
processScore = min(workLogScore, 30) + min(classPresentationScore, 30) + min(peerReviewScore, 40)

// AI追踪活跃度（0-100）
aiActivity = min(100, (loginCount * 2 + activeMinutes / 10) / 2)
// 若 aiActivity < 60，触发低活跃预警
```

### 3.2 组内自评贡献分校验
```
// 3人组总分池≤42, 4人组≤56, 5人组≤70
maxPool = memberCount * 14
// 组长分配各组员贡献分，系统校验总和不超过maxPool
```

### 3.3 个人分自动计算
```
personalScore = (contributionScore / groupTotalPool) * 100 * 0.7 + average(peerScores) * 0.3
```

### 3.4 结果性评价计算
```
// 单任务得分
resultScore = groupScore * 0.6 + personalScore * 0.4

// 最终结果性评价（目前按单任务计算）
finalResultScore = groupTotal * 0.6 + personalScore * 0.4
```

### 3.5 发展性评价
```
devScore = realBreakthrough + techBreakthrough + management  // ≤10
```

### 3.6 总评计算
```
totalScore = processScore * 0.4 + resultScore * 0.6 + devScore
// 安全否决优先
if (safetyVeto) totalScore = min(59, totalScore)
```

### 3.7 增值评价计算（核心）
```
// 后测值自动计算（每项能力）
postTestValue = preTestValue * 0.2 + processRelated * 0.3 + resultRelated * 0.5

// 映射规则:
// - 应急方案制定 → 过程性:workLog + 结果性:planResult
// - 航线规划     → 过程性:workLog + 结果性:practical  
// - 设备运维     → 过程性:aiTracking + 结果性:practical
// - 飞行操控     → 过程性:classPresentation + 结果性:practical
// - 团队协作     → 过程性:peerReview + 结果性:groupScore
// - 安全意识     → 过程性:workLog + 结果性:practical (加权)

// 绝对增值
absoluteGrowth = postTestValue - preTestValue

// 相对增值排名（百分位）
relativeRank = (比该学生增值低的人数 / 总人数) * 100
```

- **过程性评价面板**: 工单日志分数、课堂展示分数、AI分析按钮、组内自评贡献分输入

## 5. 模拟数据规格

G1: "雄鹰组", G2: "飞龙组", G3: "猎鹰组", G4: "天翼组", G5: "云豹组", G6: "雷鹰组"

- 前测能力值范围: 40-80分（随机但固定种子）

- 触发所有图表刷新

## 6. 交互规格

### 6.1 角色切换
- 点击顶部"教师"/"学生"按钮切换
- 切换后立即刷新侧边栏和主区域
- 学生视角只能看到自己和本组数据

### 6.2 学生选择
- 教师视角: 点击侧边栏学生项，主区域显示该学生数据
- 学生视角: 默认选中当前登录学生

### 6.3 数据编辑与联动
- 任何输入框/滑块变化 → 立即重新计算 → 更新所有显示
- 图表使用ECharts setOption更新，无刷新
- localStorage每5秒或关键操作后自动保存

### 6.4 AI模拟功能
- "AI演讲分析"按钮: 生成随机六维雷达图数据并展示
- "AI预警检查"按钮: 扫描所有学生，生成预警事件列表
- 所有AI功能纯前端模拟
